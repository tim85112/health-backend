	資料表格(Table)命名規則

	首字為英文字母
	不可加入空白、中文字、除了 "_" 外其它符號
	全部小寫 / 全部大寫 / 首字大寫
	統一採用 單數 / 複數 英文單字
	使用 模組 分類

	欄位(Field) 命名規則

	首字為英文字母
	不可加入空白、中文字、除了 "_" 外其它符號
	全部小寫 / 全部大寫 / 首字大寫
	使用 全名 / 簡寫
	使用 欄位分類
	分類 後置 / 前置
	主檔 及 表單 使用 特定名稱
